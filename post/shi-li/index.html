Ubuntu 20.04 修复 Dash to dock

缘起

Dash to dock 是大家都很喜欢使用的插件，但是现在它在 Ubuntu 20.04 下经常出现各种问题，最经典的就是莫名其妙出现双重dock。 网上总说您没有关闭 Ubuntu Dock，一打开优化一看，阿好，没开呢！于是乎GG。

修复

几经周折，在Github的一只Issue里面找到了解决方案。

Preparation

请确保您有： git make msgfmt。方便起见，您可以这样安装。

sudo apt install git make gettext

安装 Github 版本的 Dash to Dock

以下内容无需加速：

git clone https://github.com/micheleg/dash-to-dock.git -b gnome-3-36
cd dash-to-dock/
make
make install

重启gnome

这该会了吧…… 摁下Alt+F2快捷键，在运行那里输入r，稍后片刻即可。

现在，您应该已经看到了正常的dash to dock。快去优化中设置它吧！

鸣谢
